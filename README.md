## 下载

### Docker

```
docker pull apachecn0/guru99-zh
docker run -tid -p <port>:80 apachecn0/guru99-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install guru99-zh
guru99-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g gurujiujiu-zh
gurujiujiu-zh <port>
# 访问 http://localhost:{port} 查看文档
```
